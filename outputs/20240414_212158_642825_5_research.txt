

GitHub Copilot는 GitHub (Microsoft 소유)와 OpenAI (Microsoft가 49% 소유)가 개발한 코드 완성 도구입니다. 이 도구는 Visual Studio Code, Visual Studio, Neovim 및 JetBrains 통합 개발 환경(IDE) 사용자들에게 코드 자동완성을 지원하며 현재 개별 개발자와 기업에 구독을 통해 제공되고 있습니다. 이 인공 지능 소프트웨어는 2021년 6월 29일 첫 발표되었으며 주로 Python, JavaScript, TypeScript, Ruby, Go 언어 코딩에 가장 적합합니다. 2023년 3월에는 "Copilot X"가 발표되었으며, Copilot에는 GPT-4 기반의 챗봇과 음성 명령 지원이 포함될 예정입니다.

GitHub은 개발 플랫폼으로, 개발자들이 코드를 만들고 저장하며 관리 및 공유할 수 있게 해주며, Git 소프트웨어를 사용하여 Git의 분산 버전 관리 기능 및 액세스 제어, 버그 추적, 소프트웨어 기능 요청, 작업 관리, 지속적 통합 및 각 프로젝트에 대한 위키를 제공합니다. 캘리포니아에 본사를 두고 있으며 2018년부터 Microsoft의 자회사입니다. 오픈소스 소프트웨어 개발 프로젝트를 호스팅하는 데 자주 사용되며, 2023년 1월 기준으로 GitHub은 1억 명의 개발자 및 4억 2천만 개 이상의 저장소(최소 2천8백만 개의 공개 저장소 포함)를 보유하고 있으며, 2023년 6월 기준으로 세계에서 가장 큰 소스 코드 호스트입니다.

GPT-4는 OpenAI가 만든 다중모드 대형 언어 모델로, GPT 시리즈의 네 번째에 속합니다. 2023년 3월 14일에 출시되었으며, 유료 챗봇 제품인 ChatGPT Plus, OpenAI의 API 및 무료 챗봇 Microsoft Copilot을 통해 공개적으로 사용할 수 있게 되었습니다. 트랜스포머 기반 모델인 GPT-4는 공개 데이터와 "제3자 제공자로부터 라이센스를 받은 데이터"를 사용한 사전 훈련을 통해 다음 토큰을 예측하는 패러다임을 사용하고, 이후 사람과 AI의 강화 학습 피드백을 사용하여 인간 정렬 및 정책 준수를 위한 세부 조정이 이루어졌습니다._observer가 GPT-3.5를 기반으로 한 이전 버전보다 개선된 것으로 보고했으나, GPT-4는 여전히 이전 버전들과 비슷한 문제를 유지하고 있다고 합니다. GPT-4는 GPT-4V라 불리는 시각 능력을 갖추고 있으며, ChatGPT에서 이미지를 입력으로 사용할 수 있습니다. OpenAI는 GPT-4의 구체적인 기술적 세부 사항 및 통계, 예를 들어 모델의 정확한 크기와 같은 다양한 정보를 공개하는 것을 거부했습니다.
