

CPython은 Python 프로그래밍 언어의 참조 구현입니다. C와 Python으로 작성된 CPython은 Python 언어의 기본적이고 가장 널리 사용되는 구현입니다. CPython은 Python 코드를 바이트코드로 컴파일한 다음 해석하기 때문에 인터프리터이자 컴파일러로 정의될 수 있습니다. 여러 언어와의 외국어 기능 인터페이스를 가지고 있으며, Python이 아닌 언어로 명시적으로 바인딩을 작성해야 하는 C와 같은 언어들이 여기에 포함됩니다.
